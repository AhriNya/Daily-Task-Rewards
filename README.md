Bing Auto-Search & Reading Auditor (V48.14)
一个专为 Bing 打造的高级自动化搜索与模拟审计工具。采用极致的 Glassmorphism（毛玻璃）设计语言，结合拟人化操作逻辑，在提升搜索活跃度的同时，提供丝滑的用户体验。

✨ 核心特性
🎨 极致视觉体验
Frosted Glass UI: 深度高斯模糊面板，支持实时背景透视与 180% 饱和度增强。

隐形挂件模式: 任务未启动时，入口按钮以 90% 透明度（0.1 opacity）静默存在，完全不干扰网页浏览；悬停即刻唤醒。

毫秒级状态看板: 倒计时精确至 0.1s，配合等宽字体设计，数字跳动平稳不抖动。

🤖 拟人化行为引擎
智能打字仿真: 模拟真人逐字输入，随机间隔时间，规避机械检测。

深度阅读审计: 自动模拟网页平滑滚动，模拟人类翻阅搜索结果的行为。

随机任务链路: 随机的阅读时长 (15-30s) 与 随机的冷却间隔 (5-15min)，让自动化轨迹更自然。

🧠 智能逻辑管控
每日首启保护: 每天第一次打开网页默认不开启任务，由用户手动点亮“第一把火”，之后全天自动记忆状态。

不重复搜索: 拥有 2000 条容量的“历史账本”，自动过滤已搜关键词。

全中文文案定制: 逻辑位置与显示文字完全解耦，支持一键自定义 I18N 文字。

🛠️ 技术实现
存储方案: 使用 GM_setValue/getValue 实现跨页面、跨session的状态持久化。

数据来源: 动态抓取百度实时热搜接口，确保关键词的时效性。

UI 引擎: 原生 CSS backdrop-filter + 异步 DOM 渲染。

🚀 快速开始
安装 Tampermonkey 浏览器扩展。

新建脚本，将 Bing_Auto_Search.js 的代码粘贴进去。

打开 Bing，点击左下角几乎透明的小圆点即可启动。
